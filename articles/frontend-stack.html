前端技术栈速览：
Vue 3 · Tailwind · daisyUI · Croppie

2026-01-27 前端资料

栈组合概要

Vue 3 作为核心框架，Composition API + 更佳 TS 支持。

Tailwind CSS 原子化样式，直接写 class；daisyUI 作为 Tailwind 插件补齐按钮 / 卡片 / Modal 等 UI 组件与主题。

Croppie 处理头像裁剪等上传前预览，直接用原生版本即可。

推荐工作流：Vite + Tailwind + daisyUI + Vue 3 组件化，按需集成 Croppie

核心框架：Vue 3

Vue 3 提供组合式 API、更优 TS 体验与更高性能，官方推荐使用 Vite 构建。快速上手步骤：

  1. 确保已安装 Node.js ^20.19.0 || >=22.12.0。
  2. 创建项目并按提示选择特性：
npm create vue@latest
cd <project-name>
npm install
npm run dev

无需构建的场景可用 CDN 直接挂载：

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<div id="app">{{ message }}</div>
<script>
  const { createApp, ref } = Vue
  createApp({
    setup() {
      const message = ref('Hello Vue!')
      return { message }
    }
  }).mount('#app')
</script>
  • 文档：快速上手（官方中文）。
  • 推荐 IDE：VS Code + 官方 Vue - Official 扩展。

样式系统：Tailwind CSS

Tailwind 扫描模板类名生成样式，零运行时，最顺滑的组合是直接作为 Vite 插件使用。

  1. 在项目根创建 Vite 项目或复用现有项目。
  2. 安装插件：
npm install tailwindcss @tailwindcss/vite
  1. 在 vite.config.ts/.js 中启用插件：
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [tailwindcss()],
})
  1. 在样式文件（如 src/style.css）引入：
@import "tailwindcss";
  1. 运行开发服务器并开始使用原子类：
npm run dev

官方指引：Using Vite。

UI 组件：daisyUI

daisyUI 基于 Tailwind，提供 Button / Card / Modal / Form 等 60+ 组件与主题系统，装好 Tailwind 后直接作为插件使用。

  1. 安装插件：
npm i -D daisyui@latest
  1. 在主样式中启用：
@import "tailwindcss";
@plugin "daisyui";

更多框架安装示例与组件清单： 安装文档 · 组件列表

图片裁剪：Croppie

轻量图片裁剪库，适合头像、上传前处理。Vue 3 可直接用原生 Croppie；需要旋转/缩放/圆形裁剪时开启相应选项。

  1. 安装或直接引用静态资源：
npm install croppie
<link rel="stylesheet" href="croppie.css" />
<script src="croppie.js"></script>
  1. 基础用法（原生或 jQuery）：
// 原生
const el = document.getElementById('crop');
const croppie = new Croppie(el, {
  viewport: { width: 200, height: 200, type: 'circle' },
  boundary: { width: 320, height: 320 },
  enableExif: true,
});

// 绑定图片
croppie.bind({ url: '/avatar.jpg' });

// 导出结果
const blob = await croppie.result({ type: 'blob', format: 'png', size: 'viewport' });
  • 常用选项：enableExif（读取方向）、enableZoom/enableResize、viewport（宽高与类型）。
  • 文档与 Demo：官方站点。

组合建议

  • 快速 MVP：Vue 3 + Tailwind + daisyUI，零设计成本；Croppie 按需引入。
  • 设计系统：daisyUI 主题能力 + Tailwind 原子化，先确定配色与字号 token，再抽象组件。
  • 部署/生产：Vite 构建，npm run build 产物发布；CDN 模式仅适合静态增强或 Demo。
